# Search Engine Submission

Use this checklist after deployment to submit Fraudly to major search engines.

## Sitemap URL

- `https://fraudly.app/sitemap.xml`

## 1) Google Search Console

1. Open [Google Search Console](https://search.google.com/search-console).
2. Add and verify the property for `https://fraudly.app`.
3. Go to **Sitemaps** and submit:
   - `https://fraudly.app/sitemap.xml`
4. Use **URL Inspection** and request indexing for:
   - homepage (`https://fraudly.app/`)
   - features page (`https://fraudly.app/features`)
   - key domain/result pages (for example `https://fraudly.app/check/example.com`)

## 2) Bing Webmaster Tools

1. Open [Bing Webmaster Tools](https://www.bing.com/webmasters/).
2. Add and verify the site property.
3. Submit sitemap:
   - `https://fraudly.app/sitemap.xml`
4. Optionally use URL submission tools for priority pages.

## 3) Yandex Webmaster (optional)

1. Open [Yandex Webmaster](https://webmaster.yandex.com/).
2. Add and verify the site.
3. Submit sitemap:
   - `https://fraudly.app/sitemap.xml`

## Quick verification

- Check `https://fraudly.app/robots.txt` and confirm sitemap is listed.
- Re-submit sitemap after major route/content updates.
